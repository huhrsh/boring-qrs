# Example QR Codes

Upload your 3 example QR code images here with these exact filenames:

1. **qr-portrait.png** or **qr-portrait.jpg** - Portrait/people QR example
2. **qr-landscape.png** or **qr-landscape.jpg** - Landscape/scenic QR example  
3. **qr-brand.png** or **qr-brand.jpg** - Brand/marketing QR example

Recommended image specs:
- Format: PNG (preferred) or JPG
- Size: 800x800px minimum (square)
- File size: Under 500KB for fast loading

These images will be displayed in the "What You Can Create" gallery section on the homepage.
